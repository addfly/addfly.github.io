# 🐶第一章 HTML简介

什么是 HTML？HTML 是用来描述网页的一种语言。

  • HTML 指的是超文本标记语言 (Hyper Text Markup Language)
  • HTML 不是一种编程语言，而是一种标记语言 (markup language)
  • 标记语言是一套标记标签 (markup tag)
  • HTML 使用标记标签来描述网页

# 🐰第五章 表单

form

属性名 描述
action 提交页面URL
enctype 编码方式（默认url-encoded）
method 提交表单时用的HTTP方法（默认: GET）
name 识别表单的名称
target 规定 action 属性中地址的目标（默认：_self）

常用的表单元素：（下列内容不含HTML5新增内容）

  • <input> ：根据不同的 type 属性，可以变化为多种形态。
    • type="text" ：定义供文本输入的单行输入字段
    • type="password" ：定义密码字段
    • type="submit" ：定义提交表单数据至表单处理程序的按钮
    • type="radio" ：定义单选按钮
    • type="checkbox" ：定义复选框
    • type="button" ：定义按钮
    • value 属性 ：规定输入字段的初始值
    • readonly 属性 ：规定输入字段为只读
    • disabled 属性 : 规定输入字段是禁用的
    • size 属性 ：规定输入字段的尺寸（以字符计）
    • maxlength 属性 ：规定输入字段允许的最大长度
  • <select> ：定义下拉列表
    • <option> ：定义待选择的选项
  • <textarea> ：定义多行输入字段（文本域）
  • <button> ：可点击的按钮

表单输入限制

属性 描述
disabled 规定输入字段应该被禁用。
max 规定输入字段的最大值。
maxlength 规定输入字段的最大字符数。
min 规定输入字段的最小值。
pattern 规定通过其检查输入值的正则表达式。
readonly 规定输入字段为只读（无法修改）。
required 规定输入字段是必需的（必需填写）。
size 规定输入字段的宽度（以字符计）。
step 规定输入字段的合法数字间隔。
value 规定输入字段的默认值。
