# box-shadow ：CSS3 盒阴影

语法：

.div{
    box-shadow: h-shadow v-shadow blur spread color inset;
}
1
2
3
值 说明
h-shadow 必需的。水平阴影的位置。允许负值
v-shadow 必需的。垂直阴影的位置。允许负值
blur 可选。模糊距离
spread 可选。阴影的大小
color 可选。阴影的颜色。在CSS颜色值寻找颜色值的完整列表
inset 可选。从外层的阴影（开始时）改变阴影内侧阴影即阴影反向嵌入图形中

# border-image ：CSS3 边界图片(是一个简写属性)

用于设置以下属性：

  • border-image-source ：用在边框的图片的路径。
  • border-image-slice ： 切割属性，将图片切割为9部分，切割线按上右下左（可缩写）规则，每部分到相应位置。
  • border-image-width ：图片边框的宽度。
  • border-image-outset ：边框图像区域超出边框的量。
  • border-image-repeat ：图像边框是否应平铺(repeated)、铺满(rounded)或拉伸(stretched)。

语法：

.div{
    border-image: source slice width outset stretch|repeat|round|initial|inherit;
}

# 🐴第四章 CSS3 渐变

CSS3 定义了两种类型的渐变（gradients）：

  • 线性渐变（Linear Gradients）- 向下/向上/向左/向右/对角方向
  • 径向渐变（Radial Gradients）- 由它们的中心定义

# 1.线性渐变的方向控制

  • 从上到下（默认）
#div1 {
    background-image: linear-gradient(red, yellow, green);
}
1
2
3
  • 从左到右
#div2 {
    background-image: linear-gradient(to right,red, yellow, green);
}
1
2
3
  • 对角
 #div3 {
    background-image: linear-gradient(to top right,red, yellow, green);
}
1
2
3
  • 角度（deg）

可用负数，0deg 将创建一个从下到上的渐变，90deg 将创建一个从左到右的渐变。

WARNING

旧版本浏览器，使用旧的标准，即 0deg 将创建一个从左到右的渐变

# 2.渐变的颜色控制

  • 可用百分比来划分颜色比例
#div1 {
    width: 200px;
    height: 100px;
    background-image: linear-gradient(red 10%, yellow 70%, green 90%);
}

    width: 200px;
    height: 100px;
    background-image: linear-gradient(rgba(255,0,0,0),rgba(255,0,0,1),yellow);

# 🐝第六章 CSS3 字体

在 CSS3 之前，web 设计师必须使用已在用户计算机上安装好的字体。通过 CSS3，web 设计师可以使用他们喜欢的任意字体，可将该字体文件存放到 web 服务器上，它会在需要时被自动下载到用户的计算机上。

CSS3 @font-face 规则

在新的 @font-face 规则中，您必须首先定义字体的名称（比如 myFirstFont），然后指向该字体文件。如需为 HTML 元素使用字体，请通过 font-family 属性来引用字体的名称 (myFirstFont)

<style>
    @font-face {
        font-family: myFirstFont;
        src: url(sansation_light.woff)
    }

    div {
        font-family: myFirstFont;
    }
</style>

# 🐛第七章 CSS3 2D 转换

使用transform:方法完成 2D 转换，也改变元素基点transform-origin

# translate()

根据左( X 轴)和顶部( Y 轴)位置给定的参数，从当前元素位置移动

# rotate()

在一个给定度数顺时针旋转的元素。负值是允许的，这样是元素逆时针旋转

# scale()

该元素增加或减少的大小，取决于宽度（ X 轴）和高度（ Y 轴）的参数

# skew()

包含两个参数值，分别表示 X 轴和 Y 轴倾斜的角度，如果第二个参数为空，则默认为0，参数为负表示向相反方向倾斜

skew可以简单理解为对x取值就是横向左右拉伸，对y取值是上下拉伸

也可以理解为在skew(xdeg,ydeg)中

  • x 取值为正，X 轴不动，y 轴逆时针倾斜一定角度
  • x 取值为负，X 轴不动，y 轴顺时针倾斜一定角度
  • y 取值为正，y 轴不动,x 轴顺时针倾斜一定角度
  • y 取值为负，y 轴不动,x 轴逆时针倾斜一定角度

图形解析理解 css3 之倾斜属性skew(), 不过对该文章的坐标系位置理解持保留意见

# matrix()

有六个参数，包含旋转，缩放，移动（平移）和倾斜功能

矩阵依靠线性代数知识 对CSS3中的transform：Matrix（）矩阵的一些理解

# 🐚第十章 CSS3动画

# @keyframes 规则

@keyframes 规则是创建动画。

@keyframes 规则内指定一个 CSS 样式和动画将逐步从目前的样式更改为新的样式。

<head>
    <style>
        @keyframes changColor {
            /* 可以用百分比规划时间或 from，to 的组合*/
            from {
                background: red;
            }
            to {
                background: green;
            }
        }
    </style>
</head>
<body>
    <div style="animation: changColor 3s;width:100px;height:100px;background-color: green;"></div>
</body>

# 🐜第十二章 CSS3 用户界面

在 CSS3 中, 增加了一些新的用户界面特性来调整元素尺寸，框尺寸和外边框。

# resize:both 属性

指定一个元素是否应该由用户去调整大小

可选 描述
none 用户无法调整元素的尺寸。
both 用户可调整元素的高度和宽度。
horizontal 用户可调整元素的宽度。
vertical 用户可调整元素的高度

WARNING

需要同时设置overflow属性

# box-sizing 属性

以确切的方式定义适应某个区域的具体内容。

可选 描述
content-box(默认值) 设置的宽度即内容宽度
border-box 对元素指定宽度和高度包括了 padding 和 border 。

信息

box-sizing 最主要的用法是规定容器元素的最终尺寸计算方式，防止得到比预期更大的容器从而破坏网页布局

# outline-offset 属性

对轮廓进行偏移，并在超出边框边缘的位置绘制轮廓

轮廓与边框有两点不同：

  • 轮廓不占用空间
  • 轮廓可能是非矩形
div{
    border:1px solid black;
    outline:1px solid red;
    /* 在边框之外10像素处有一个轮廓 */
    outline-offset:10px;
}

# 🐍第十五章 CSS3 多媒体查询

CSS3 的多媒体查询继承了 CSS2 多媒体类型的所有思想： 取代了查找设备的类型

媒体查询可用于检测很多事情，例如：

  • viewport(视窗) 的宽度与高度
  • 设备的宽度与高度
  • 朝向 (智能手机横屏，竖屏) 。
  • 分辨率

# 媒体查询语法

@media not|only mediatype and (expressions) {
    /* CSS 代码...; */
}
1
2
3
  • not: not是用来排除掉某些特定的设备的，比如 @media not print（非打印设备）。
  • only: 用来定某种特别的媒体类型。对于支持Media Queries的移动设备来说，如果存在only关键字，移动设备的Web浏览器会忽略only关键字并直接根据后面的表达式应用样式文件。对于不支持Media Queries的设备但能够读取Media Type类型的Web浏览器，遇到only关键字时会忽略这个样式文件。
  • all: 所有设备，这个应该经常看到。

也可以在不同的媒体上使用不同的样式文件

<link rel="stylesheet" media="mediatype and|not|only (expressions)" href="print.css">
1
  • all 用于所有多媒体类型设备
  • print 用于打印机
  • screen 用于电脑屏幕，平板，智能手机等。
  • speech 用于屏幕阅读器

# 媒体功能

值 描述
max-height 定义输出设备中的页面最大可见区域高度。
max-width 定义输出设备中的页面最大可见区域宽度。
width 定义输出设备中的页面可见区域宽度。
height 定义输出设备中的页面可见区域高度。
min-height 定义输出设备中的页面最小可见区域高度。
min-width 定义输出设备中的页面最小可见区域宽度。
resolution 定义设备的分辨率。如：96dpi, 300dpi, 118dpcm
