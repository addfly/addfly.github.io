# Flex布局

布局的传统解决方案，基于盒状模型，依赖 display 属性 + position 属性 + float 属性。它对于那些特殊布局非常不方便，比如，垂直居中就不容易实现。2009年，W3C 提出了一种新的方案----Flex 布局，可以简便、完整、响应式地实现各种页面布局。

WARNING

本部分参考自阮一峰的网络日志，详细请前往Flex 布局教程：语法篇

# 🐤第一章 简介

Flex 是 Flexible Box 的缩写，意为"弹性布局"，用来为盒状模型提供最大的灵活性。

# 1.简单使用

任何一个容器可以指定为 Flex 布局。

.box{
  display: flex;
}
1
2
3

行内元素也可以使用 Flex 布局

.box{
  display: inline-flex;
}

2
3

提示

设为 Flex 布局以后，子元素的 float 、 clear 和 vertical-align 属性将失效。

# 2.基本概念

借用一张图来解释flex布局内部结构

  • 应用flex的元素称为容器，子元素自动成为容器成员，称为flex项目，简称项目。
  • 容器上有两根轴主轴( main )和交叉轴( cross )。

# 🦆第二章 容器属性

属性名 描述 默认值 可选值
flex-direction 主轴方向 row row-reverse,column,column-reverse
flex-wrap 如果一条轴线排不下，如何换行 nowrap wrap,wrap-reverse
flex-flow 主轴方向和换行合并写法 row nowrap <flex-direction> <flex-wrap>
justify-content 主轴对齐方式 flex-start flex-end,center,space-between,space-around
align-items 交叉轴对齐方式 stretch flex-start,flex-end,center,baseline
align-content 多轴线对齐方式 stretch flex-start,flex-end,center,space-between,space-around

# 🦅第三章 项目的属性

属性名 描述 默认值 可选值
order 项目的排列顺序，数值越小，排列越靠前 0 数值
flex-grow 定义项目的放大比例，数值越大，占用的比例越多 0 数值
flex-shrink 定义了项目的缩小比例，默认为1，即如果空间不足，该项目将缩小 1 0不缩小
flex-basis 项目占据的主轴空间 auto 长度
flex none <'flex-grow'> <'flex-shrink'> <'flex-basis'>
align-self 允许单个项目有与其他项目不一样的对齐方式，可覆盖align-items属性 auto flex-start,flex-end,center,baseline,stretch
