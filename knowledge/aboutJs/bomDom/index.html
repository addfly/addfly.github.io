# 🍑第一章 BOM（浏览器对象模型）

BOM提供了很多对象，用于访问浏览器的功能。BOM的核心对象是window，它表示浏览器的一个实例。在浏览器中，window对象有双重角色，它既是通过JavaScript访问浏览器窗口的一个接口，又是ECMAScript规定的Global对象。

WARNING

BOM根据浏览器版本及内核不同，对方法或属性有不同支持。

# 1.全局作用域

如果在全局环境中用var定义了对象、变量或函数都会被添加到window对象中。

  • 全局变量不能通过delete操作符删除，而直接在window上定义的属性可以
  • 使用var添加的window属性有一个名为 [[configurable]] 特性，值为false
  • 尝试访问未声明的变量会抛出错误，但是通过查询window对象，会成为属性查询，值为undefined

# 2.浏览器窗口位置

浏览器窗口位置属性：

  • screenLeft
  • scrollTop
  • scrollX
  • scrollY

浏览器窗口移动方法(大部分浏览器禁用此方法)：

  • window.moveTo(x,y) ：移动到(x,y)
  • window.moveBy(x,y) ：移(x,y)

# 3.浏览器窗口大小

浏览器窗口大小属性：

属性名 描述
innerWidth 浏览器视图区宽度
innerHeight 浏览器视图区高度
outerWidth 浏览器窗口宽度
outerHeight 浏览器窗口高度

浏览器窗口大小操作方法（大部分浏览器禁用）：

方法名 参数 描述
resizeTo() (新宽度，新高度) 调整到某大小
resizeBy() (新旧宽度差，新旧高度差) 调整某大小

# 4.导航和打开窗口(open,close)

window.open()用于打开一个新的浏览器窗口或查找一个已命名的窗口。

语法：window.open([URL[,name[,specs[,replace)

window.close()：关闭浏览器窗口。

# ⛳️5.间歇调用和超时调用

  • 超时调用(在指定时间后调用)是setTimeout()方法。
  • 间歇调用(每隔一定时间就执行一次代码)是setInterval()方法。

在使用时会返回间歇调用ID或超时调用ID

用 clearTimeout() 方法和 clearInterval() 方法可以取消间歇调用或超时调用

# 6.系统对话框（alert,confirm,prompt）

方法名 参数 描述
alert() 指定消息 显示带有一条指定消息和一个确认按钮的警告框
confirm() 指定消息 显示一个带有指定消息和确认及取消按钮的对话框，如果访问者点击"确定"，此方法返回true，否则返回false。
prompt() （[msg [，defaultText） 用于显示可提示用户进行输入的对话框。这个方法返回用户输入的字符串。
print() 方法用于打印当前窗口的内容。
  • msg:要在对话框中显示的纯文本
  • defaultText:默认的输入文本

TIP

系统对话框外观由操作系统及浏览器决定。

通过这些方法打开的对话框都是同步和模态的（即在显示对话框时代码会停止执行）

# ⛳️7.location对象属性

location对象是很特别的一个对象，因为它既是window对象的属性，也是document对象的属性，它们引用的是同一个对象，location对象可以获取当前url地址，也可以将当前的url地址解析成独立的片段。

属性名 例子 说明
hash "#contents" 返回url的hash（返回一个URL的锚部分）
host "www.location.com:8080" 返回一个URL的主机名和端口
hostname "www.location.com" 返回URL的主机名
href "https://www.location.com:8080/user/" 返回完整的URL
pathname "/user/" 返回的URL路径名。
port "8080" 返回一个URL服务器使用的端口号
protocal "https" 返回一个URL协议
search "?name=zhangsan" 返回一个URL的查询部分
// 正常情况获取查询参数（不去除问号）
 //"?name=%E5%BC%A0%E4%B8%89&age=20"
let querySearch = location.search.length > 0 ? location.search : ""

//查询字符串参数扩展
// 访问URL包含的查询字符串属性并不太方便，无法逐个访问其中的每个查询字符串参数
// 可以用以下工具函数
let searchMessage = "?name=%E5%BC%A0%E4%B8%89&age=20"
let getSearchMessage = (value) => {
    // 保存转换后的对象 
    let obj = {};
    // 存放根据&分隔符分割的字符串
    let args = [];
    args = value.substring(1).split('&');
    let v = args.forEach((item) => {
        var arr = [],
            name = "",
            value = "";
        arr = item.split("=");
        // 解码
        name = decodeURIComponent(arr[0]);
        value = decodeURIComponent(arr[1]);
        // 添加到obj中
        obj[name] = value;
    });
    console.log(obj);
};
getSearchMessage(searchMessage);

# 8.Location 对象方法

方法 参数 说明
assign() URL 载入一个新的文档
reload() [forceGet)可选，详细见下方tip 重新载入当前文档，类似于你浏览器上的刷新页面按钮
replace() newURL 用新的文档替换当前文档（差别详见下方tip）
window.location.assign("https://baidu.com")
1

TIP

  1. 如果把 reload() 方法的参数设置为 true，那么无论文档的最后修改日期是什么，它都会绕过缓存，从服务器上重新下载该文档。这与用户在单击浏览器的刷新按钮时按住 Shift 健的效果是完全一样。

  2. location.replace() 方法可用一个新文档替代当前文档。但是与 location.assign() 和location.href 不同的是，此方法是用指定URI替换当前URI。所以被替换的URI不会被保存的历史记录中，所以点击浏览器后退按钮不会返回被替换URI。

# 9.navigator对象

navigator 对象已经成为识别客户端浏览器的事实标准，主要用于检测浏览器相关信息。没有应用于 navigator 对象的公开标准，不过所有浏览器都支持该对象。

navigator对象的属性 说明
appCodeName 返回浏览器的代码名
appName 返回浏览器的名称
appVersion 返回浏览器的平台和版本信息
cookieEnabled 返回指明浏览器中是否启用 cookie 的布尔值
onLine 是否联网
platform 返回运行浏览器的操作系统平台
userAgent 返回由客户机发送服务器的user-agent 头部的值

# 10.screen对象

Screen 对象包含有关客户端显示屏幕的信息。

screen对象的属性 说明
availHeight 返回屏幕的高度（不包括Windows任务栏）
availWidth 返回屏幕的宽度（不包括Windows任务栏）
colorDepth 返回目标设备或缓冲器上的调色板的比特深度

width 返回屏幕的总宽度

# 11.history对象

History 对象包含用户（在浏览器窗口中）访问过的 URL。 History 对象是 window 对象的一部分，可通过 window.history 属性对其进行访问。

History属性或方法 说明
length 返回历史列表中的网址数
History 对象方法
back() 加载 history 列表中的前一个 URL
forward() 加载 history 列表中的下一个 URL
go() 加载 history 列表中的某个具体页面

# ☎️第二章 DOM（文档对象模型）

文档对象模型 (DOM) 将 web 页面与到脚本或编程语言连接起来。通常是指 JavaScript，但将 HTML、SVG 或 XML 文档建模为对象并不是 JavaScript 语言的一部分。DOM模型用一个逻辑树来表示一个文档，树的每个分支的终点都是一个节点(node)，每个节点都包含着对象(objects)。DOM的方法(methods)让你可以用特定方式操作这个树，用这些方法你可以改变文档的结构、样式或者内容。节点可以关联上事件处理器，一旦某一事件被触发了，那些事件处理器就会被执行。

概念性的DOM介绍

# 1. Node节点关系

Node 是一个接口，许多 DOM API 对象的类型会从这个接口继承。

属性或方法 说明
nodeType属性 表示节点类型
nodeName属性 返回一个包含该节点名字的DOMString。如Text节点对应的是 '#text'
nodeValue属性 返回或设置当前节点的值
childNodes属性 返回一个包含了该节点所有子节点的实时的NodeList。NodeList 是“实时的”意思是，如果该节点的子节点发生了变化，NodeList对象就会自动更新
parentNode属性 返回一个当前结点 Node的父节点
previousSibling属性 返回上一个兄弟节点
nextSibling属性 返回下一个兄弟节点
ownerDocument属性 返回这个元素属于的 Document对象
hasChildNodes() 返回一个布尔值，查看是否有子节点

# 2. Node操作方法

方法 参数 说明
appendChild() 将指定的 childNode 参数作为最后一个子节点添加到当前节点。如果参数引用了 DOM 树上的现有节点，则节点将从当前位置分离，并附加到新位置。
insertBefore() （插入节点，参考节点） 在当前节点下增加一个子节点 Node，并使该子节点位于参考节点的前面。
replaceChild() （要插入的节点，要替换的节点） 替换节点
removeChild() 要移除的节点节点 移除节点

TIP

  • cloneNode() ：克隆一个 Node，并且可以选择是否克隆这个节点下的所有内容。默认情况下，节点下的内容会被克隆。可以加入一个布尔值参数
  • normalize() ：将当前节点和它的后代节点”规范化“（normalized）。在一个"规范化"后的DOM树中，不存在一个空的文本节点，或者两个相邻的文本节点。

# 3.Document类型

JavaScript类型表示文档，在浏览器中，document对象是HTMLDocument的一个实例，表示整个HTML对象。而且document对象是window对象的一个属性，因此可以将其作为全局对象来访问

文档的子节点

属性 说明
documentElement 返回当前文档的直接子节点。对于 HTML 文档，HTMLHtmlElement 对象一般代表该文档的<html> 元素。
body 返回文档的<body>节点
doctype 返回当前文档的文档类型定义

文档信息

属性 说明
title 文档标题，可修改
URL 字符串形式返回文档的地址栏链接
domain 取得域名
referrer 来源页面的 URI

查找元素

方法 说明
getElementById() 根据id
getElementsByTagName() 根据标签名
getElementsByName() 根据name特性

文档写入

方法 说明
write() 接受字符串参数，原样写入，会重写整个页面内容
writeln() 接受字符串参数，末尾换行符（\n），会重写整个页面内容
open() 打开网页输出流
close() 关闭网页输出流

# 4.Element类型

Element 是一个通用性非常强的基类，所有 Document 对象下的对象都继承自它。这个接口描述了所有相同种类的元素所普遍具有的方法和属性。

属性 说明
id
title
className
dir
lang

操作特性的DOM方法

方法 参数 说明
getAttribute() 特性名 获取
setAttribute() （特性名，值） 设置
removeAttribute() 特性名 移除

attributes属性(没上方的三个方法好用)

attributes属性的方法 参数 说明
getNamedItem() name 返回nodeName等于name的节点
removeNamedItem() name 移除nodeName等于name的节点
setNamedItem() node 向列表中添加节点，以nodeName属性为索引

创建元素

使用 document.createElement() 方法创建新元素，接收一个参数，即要创建元素的标签名

WARNING

元素的子节点中空白符会占用节点位置
